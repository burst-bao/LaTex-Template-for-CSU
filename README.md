# LaTex-Template-for-CSU（非官方）

## 1.LaTex Template for CSUBachelor

在武汉大学本科毕业论文模板的基础上，根据中南大学本科毕业论文的格式要求，进行删改，较为完美地实现了中南大学本科生毕业论文word模板中罗列的所有要求。 

内包含中南大学本科毕业设计论文模板以及IEEE论文翻译模板。
